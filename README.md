# FRAME — Portfólio Audiovisual

Portfólio audiovisual desenvolvido para a marca **FRAME**, com proposta visual cinematográfica, layout premium e foco em apresentar vídeos, fotografia e direção visual de forma profissional.

## Preview

Projeto criado para funcionar como um site de apresentação da marca, com:
- introdução animada
- vídeo de fundo
- seções com projetos
- serviços
- contato
- visual responsivo

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Conceito do projeto

A proposta da FRAME é transmitir:
direção visual
presença de marca
estética forte
acabamento profissional
linguagem audiovisual moderna
O site foi desenvolvido com uma identidade visual minimalista, usando preto, branco e tons de cinza, reforçando uma estética elegante e cinematográfica.

## Funcionalidades

- Intro animada com contagem regressiva
- Exibição da identidade da marca antes da abertura do site
- Vídeo de fundo em tela cheia
- Hero principal com destaque visual
- Seção de apresentação da FRAME
- Área de projetos com vídeos em destaque
- Seção de serviços
- Área de contato
- Layout responsivo para diferentes tamanhos de tela

## Objetivo

- Este projeto foi desenvolvido para servir como: portfólio (OBS: Tinha a frame como um projeto meu no audivisual que meio que deixei de lado para seguir como Desenvolvedor)
- apresentação visual profissional
- base para futuros clientes e projetos
- vitrine digital para serviços audiovisuais

## Como executar localmente

Basta abrir o arquivo index.html no navegador.
Para uma experiência melhor durante o desenvolvimento, recomenda-se usar a extensão Live Server no VS Code.

## Responsividade

O site foi estruturado para funcionar em:
- desktop
- tablet
- mobile
